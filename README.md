# 💰💼 Финансовый Telegram-бот - помощник

Современный, аккуратный и удобный Telegram-бот на Python для учебного проекта по теме управления личными финансами.

---

## ✨🚀 Кратко о проекте

Этот проект создан как понятный, удобный и современный учебный Telegram-бот, который помогает:
- 📋 Регистрация пользователей
- 💱 Получение курса валют в реальном времени
- 💡 Генерация советов по экономии
- 📊 Учет расходов по категориям
- 🗄  Хранение данных в SQLite
- ⚡  Асинхронная работа (aiogram 3)
- 🎨 Удобный интерфейс с кнопками

### 📋 Проект специально сделан в простой структуре, чтобы его было легко:
 
- создать в PyCharm;
- понять даже новичку;
- показать преподавателю;
- загрузить в GitHub;
- доработать в будущем.

## 🎯🚀 Что умеет бот

### 1. Регистрация пользователя

Пользователь нажимает кнопку регистрации, и бот сохраняет:
- `telegram_id`
- имя пользователя

### 2. Курс валют

Бот получает актуальный курс валют через API и показывает:
- 1 USD → RUB
- 1 EUR → RUB

### 3. Советы по экономии

Бот случайным образом отправляет один из заранее подготовленных советов.

### 4. Личные финансы

#### Бот по шагам спрашивает:
- 1. первую категорию расходов;
- 2. сумму по первой категории;
- 3. вторую категорию;
- 4. сумму по второй категории;
- 5. третью категорию;
- 6. сумму по третьей категории.

#### После этого бот сохраняет всё в базу и показывает красивую итоговую сводку.

### 5. Отмена ввода

Команда `/cancel` позволяет безопасно выйти из текущего процесса ввода.

- **python-dotenv** — библиотека для чтения данных из `.env`;
- **Git и GitHub** — для хранения проекта и отправки ссылки.
- **REST API**
