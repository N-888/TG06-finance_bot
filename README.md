# 💰💼 Финансовый Telegram-бот - помощник

Современный, аккуратный и удобный Telegram-бот на Python для портативного учебного проекта по теме управления личными финансами.

---

## ✨🚀 Кратко о проекте

Этот проект создан как понятный, удобный и современный учебный Telegram-бот, который производит процессы:
- 📋 Регистрация пользователей
- 💱 Получение курса валют в реальном времени
- 💡 Генерация советов по экономии
- 📊 Учет расходов по категориям
- 🗄  Хранение данных в SQLite
- ⚡  Асинхронная работа (aiogram 3)
- 🎨 Удобный интерфейс с кнопками

### 📋 Проект специально сделан в простой структуре, чтобы его было легко:
 
- создать в PyCharm;
- понять даже новичку;
- показать преподавателю;
- загрузить в GitHub;
- доработать в будущем.

## 🎯🚀 Что умеет бот

### 1. Регистрация пользователя

Пользователь нажимает кнопку регистрации, и бот сохраняет:
- `telegram_id`
- имя пользователя

### 2. Курс валют

Бот получает актуальный курс валют через API и показывает:
- 1 USD → RUB
- 1 EUR → RUB

### 3. Советы по экономии

Бот случайным образом отправляет один из заранее подготовленных советов.

### 4. Личные финансы

#### Бот по шагам спрашивает:

- первую категорию расходов;
- сумму по первой категории;
- вторую категорию;
- сумму по второй категории;
- третью категорию;
- сумму по третьей категории.

#### После этого бот сохраняет всё в базу и показывает красивую итоговую сводку.

### 5. Отмена ввода

Команда `/cancel` позволяет безопасно выйти из текущего процесса ввода.

## 🔐 Безопасность

Файл .env нельзя загружать в GitHub, потому что там лежат секретные данные:

- токен бота;
- ключ API.

Поэтому файл .env добавлен в .gitignore.

## 🗄 Где хранятся данные

Данные пользователей хранятся в файле базы данных SQLite:

user.db

В базе сохраняются:

- имя пользователя;
- Telegram ID;
- 3 категории расходов;
- 3 суммы расходов.
